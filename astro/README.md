# Astro.js BCMS Starters

Free, beautiful, customizable, responsive, high-performance **Astro.js** starters, integrated with [BCMS](https://thebcms.com/).
Built with Tailwind CSS and ready to plug into your BCMS project.

![Astro](https://img.shields.io/badge/astro-212227.svg?style=for-the-badge&logo=astro&logoColor=white) ![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white)

If you like the project, hit that ⭐ button to show some 💛

---

## Requirements

- Node.js **22.x** or newer
- npm **10.x** or newer
- Astro **6.x** (all Astro starters here are built and tested against Astro 6)

---

## Getting Started

To create a new Astro starter via the BCMS CLI:

```bash
npx @thebcms/cli create astro
```

Follow the prompts to pick the specific Astro starter you want.
